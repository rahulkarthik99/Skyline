SkylineKAI is a self-hosted, full-stack customer support starter platform that includes a real-time chat system, authentication, lead-style dashboard, and an embeddable web chat widget.

It is designed as a developer-friendly foundation that can be extended into a production-grade support system.

## Features
- Real-time chat system (WebSocket-based)
- Authentication with sessions and JWT
- Lead and CRM-style dashboard
- Embeddable website chat widget
- Responsive modern UI
- Modular and extendable architecture

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database ORM: Drizzle ORM (PostgreSQL compatible)

## Requirements
- Node.js v18 or higher
- npm
- PostgreSQL-compatible database (Neon, Supabase, local Postgres, etc.)

## Installation
1. Extract the ZIP file
2. Open a terminal in the project root
3. Run: npm install
4. Configure your database connection if required
5. Run: npm run dev
6. Visit the application in your browser

## Production
- npm run build
- npm start

## Notes
- This is a self-hosted application
- No third-party AI services are bundled by default
- Buyers may integrate their own AI providers if needed
- Intended for developers and SaaS builders
